🚀 RUN ALL TESTS (One Command):

   ./run_all_pt_tests.sh

   Generates queries + runs all tests + creates reports (~4.5 hours)

═══════════════════════════════════════════════════════════════════════

🎯 RUN SPECIFIC SCENARIO:

Multi-Collection (9 collections):
   cd multi_collection && ./run_multi_collection_all_limits.sh

Single-Collection (SongLyrics 1M):
   cd single_collection && python run_automated_tests.py

═══════════════════════════════════════════════════════════════════════

🔧 GENERATE QUERIES:

All queries (multi-collection):
   python generate_all_queries.py --type multi

All queries (single-collection):
   python generate_all_queries.py --type single

Only BM25:
   python generate_all_queries.py --type multi --search-types bm25

Only limits 50 and 100:
   python generate_all_queries.py --type multi --limits 50 100

═══════════════════════════════════════════════════════════════════════

📊 GENERATE REPORTS:

   cd report_generators
   python generate_combined_report.py    # Multi-collection
   python generate_single_report.py      # Single-collection
